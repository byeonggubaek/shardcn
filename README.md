# Monorepo 프로젝트 기본 
> Target의 Javascript 버전은 ES2022 이다.
> 개발 IDE는 Visual Studio Code를 사용한다.
> 웹서버와 DB서버는 로컬에서 테스트 한다.   

## 1. 프런트앤드 
- React 프레임워크를 이용한다.

## 2. 백앤드
- Node.js 프레임워크를 이용한다.
- DB는 Oracle을 기본으로 사용한다. 

## 3. 공통
- 공통 모듈은 가능하면 사용하지 않는것으로 한다.
- Type은 공통으로 사용할 수 있다. 
 
## 4. 개발방향
1. Shardcn UI Framwork를 사용한다. 
2. CI/CD를 테스트해본다.  
3 Docker 사용법을 테스트해본다.   
4. Github Actions을 테스트 해본다.
5. IntellJ에서 내려받아 테스트 해본다.  
6. 차후 DB는 PostgreSQL를 쓸 수 있도록 고려한다.
7. PostgreSQL DB 서버는 맥미니를 이용하여 구성할 것이다.
8. 아이맥의 아파치서버를 이용할 수도 있다.
9. 클라우드 서비스 CloudFlare, Google Cloud Free Tier, Oracle Cloud Free Tier, AWS Free Tier 사용을 테스트해 본다.     
10. 웹서버는 우선 CloudFlare를 이용한다.
11. Python 과 백앤드의 연동을 테스트한다.
12. Python에서 Machine Learning을 이용하여 AI 모델을 이용할 수 있다.
